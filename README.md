# 🎮게임 소개
1. 제목 :  **Gloomy Lamb** 
2. 장르: 3D 내러티브 어드벤쳐
3. 플랫폼: PC
4. 게임 장르: 스토리 중심의 3D 어드벤쳐
5. 세계관
이 게임의 주인공은 까만 양의 이야기이다. 하지만 어딘지 조금 칙칙해보이고 침울해보이는 건 기분 탓일까? </br>
흰 양들만 있는 세상에서 까만 양은 남들과 달라보여 어딜 가든 겉돌게 된다. 그런 까만 양에게 어느 날 마법 같은 일이 펼쳐진다.</br>
골칫거리, 말썽꾼인 까만 양은 진정한 “나 자신”을 되찾을 수 있을까?

### 조작 방법

| 이동 | W, A, S, D |

| 대시 | Left Shift |

| 스킬 1 (광선 발사) | 마우스 우클릭 |
